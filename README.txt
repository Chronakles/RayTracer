---------- Information ----------

This is a RayTracer, which reads information about a scene from an xml file and transforms it into an image using RayTracing.

---------- Instruction ----------

To run the program clone the repository, enter the executable folder and type the following command:
java -jar RayTracer.jar <examle file>

---------- tested environment ----------

The code was executed on a Windows 10 machine with exlipse JavaSE-14
The code was written on the exact same OS and development environment.
